MedBiquitous Healthcare Professional Profile Schema


Description

The MedBiquitous Healthcare Professional Profile Schema provides a data structure that allows one to represent one or more healthcare professionals in a standard format. This structure then enables the exchange of this data between related business systems and over the Web.

Schema

Version 1 of the schema file member.xsd is available at: http://ns.medbiq.org/member/v1/member.xsd

For all schema files, including the Name and Address schemas, download the zip file at: http://ns.medbiq.org/member/v1/member.zip

Specification

The schema specification is available here.

A point release version of the specification and corresponding schema are available as well.
